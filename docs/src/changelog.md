# 更新日志

## [1.0.0] - 2024-01-15

### ✨ 新增功能

- 🎉 初始版本发布
- 🧪 集成 Vitest 测试框架
- 📚 基于 VitePress 的文档站点
- 🎨 Icon 图标组件
- 🔧 完整的 TDD 开发环境

- 测试覆盖率报告

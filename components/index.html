组件 ​

Axis-UI 提供了丰富的 Vue 3 组件，支持 TypeScript 和完整的类型定义。

  • Icon 图标 - 基于字体的图标组件

🚀 快速开始 ​

安装 ​

bash
npm install @axis-ui/components
1

使用 ​

typescript

// 在组件中使用
<AxIcon :size="24" color="#3b82f6">🏠</AxIcon>

css
:root {

  --ax-color-secondary: #6b7280;
  --ax-border-radius: 4px;
